# Job Application Form Project

A responsive and modern Job Application Form built using HTML5, CSS3, and JavaScript.

# Features
- Modern UI/UX Design:
Clean layout, readable typography, responsive container, and visual hierarchy.

- Fully Responsive Layout:
Compatible with desktop, tablet, and mobile devices.

- Client-Side Form Validation:
Required fields, input structure checks, and user-friendly error messages.

- File Upload Support:
Upload CV, portfolio, or additional documents.

- Reusable CSS Structure:
Includes .container, .form-group, .header, .required classes.

- Customizable Form Sections:
Easily extendable for company requirements (position, experience, education, etc.).
